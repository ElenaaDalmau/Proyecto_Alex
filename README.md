# PROYECTO ALEX

## Metodo de uso 

clonar el repositorio
```
git clone https://github.com/ElenaaDalmau/Proyecto_Alex.git
```

acceder al repositorio
```
cd Proyecto_Alex
```

levantar el docker
```
docker-compose up --build
```

### En caso de error

editar el archivo php/index.php y modificar el siguiente campo
```
nano php/index.php

$database= coaching;
```
